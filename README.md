# Routine No. 1

> A minimalist counter for daily repetitions.

Routine No. 1 is a calm, minimalist push-up counter inspired by vintage flip clocks and intentional design. It keeps the ritual simple: count the set, make small adjustments, and return tomorrow without friction.

## Features

- Flip-clock inspired counter
- Increment and decrement controls
- Reset to zero
- Smooth number transitions
- Persistent local state
- Mobile-first design
- Minimal interface

## Getting Started

Install dependencies:

```bash
npm install
```

Start the local development server:

```bash
npm run dev
```

## Build & Deploy

Create a production build:

```bash
npm run build
```

Deploy to GitHub Pages:

```bash
npm run deploy
```

The deploy script builds the app and publishes the generated `dist` directory to GitHub Pages through the `gh-pages` package.

## Design Philosophy

Routine No. 1 is built around calm interactions, symmetry, and a tactile feel. The interface avoids the visual noise common in fitness apps, favoring intentional simplicity over dashboards, feeds, or unnecessary progress layers.

The goal is to reduce friction: open the app, record the repetitions, and leave. The counter persists locally, so the latest value remains available without accounts, setup, or external services.
